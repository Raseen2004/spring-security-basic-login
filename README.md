# Spring Security Basic Login (No JWT)

## 🔐 Features

* HTTP Basic Authentication
* Public and Private APIs
* In-memory user configuration
* Spring Security integration

## 📌 Endpoints

### Public API

```
GET /public/hello
```

No authentication required

### Private API

```
GET /private/hello
```

Requires authentication
